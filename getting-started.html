Getting Started

A quick guide to setting up and using the site — takes about two minutes to read.

📌

Saving this site for easy access

You only need to do this once. Pick whichever feels easiest:

  • Bookmark it. The simplest option — just bookmark this page in your browser like any other website.
  • Add to Home Screen / Install. Most phone browsers offer an "Install" or "Add to Home Screen" option (usually in the share menu or browser menu). This adds an icon that opens the site full-screen, without browser bars — it looks and feels like a regular app.

Either way, the icon or bookmark opens this same site — you'll still need internet access for that first tap to load it (see the note below).

⭐

Getting started with Spelling Star

  1. Open Spelling Star from the home page.
  2. The first time on a device, you'll go through a short setup: a parent PIN, your child's name, and how you want scores displayed.
  3. After setup finishes, you'll land on the child's home screen. Look for the small "Parent dashboard" link and tap it, then enter your PIN.
  4. In the parent dashboard, go to Word lists to create a list, import one from a CSV (see Resources below), or edit words directly. This is also where you'll set the weekly schedule and other settings.

You can always get back to the parent dashboard the same way later — PIN, then whatever you need to change.

🔢

Getting started with Math Star

  1. Open Math Star from the home page.
  2. The first time on a device, you'll set a parent PIN, your child's name, and a starting grade level.
  3. After setup finishes, you'll land on the child's home screen. Tap "Parent dashboard" and enter your PIN.
  4. From there you can set up focus areas (which fact categories your child practices), adjust the schedule, and check progress.

Same as Spelling Star — the "Parent dashboard" link is always there on the home screen whenever you need to make changes.

📁

Getting word lists and other files

Visit the Resources page from the home screen. Tap any word list to download it as a CSV file to your device.

To use a downloaded list in Spelling Star: open the parent dashboard → Word lists → import CSV, then select the file you downloaded.

⚠️

A word of caution: clearing your browser cache

Both apps save everything — profiles, word lists, progress, settings — directly in your browser, on that one device. There's no cloud backup.

Clearing your browsing data/cache (or "site data") for this site will erase that child's saved progress permanently. This can happen from a routine "clear browsing history" cleanup, a "free up storage" prompt, or resetting the browser. It cannot be undone.

To protect against this: use each app's Export CSV feature in the parent dashboard regularly to keep an outside copy of results. If your device ever needs a fresh start, the CSV export is your record — the app itself has no way to bring old progress back once it's cleared.

Questions or something not working? Check with whoever manages the Star Apps site.
